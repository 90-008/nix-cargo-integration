NCI Documentation

These are options that can be set under package.metadata.nix. They are package specific. They can also be set under pkgConfig under nci.lib.makeOutputs:

{
  pkgConfig = common: {
    example-package = { /* options go here */ };
  };
}

false

true

depsOverrides

dream2nix overrides for the dependency derivation of this package.

Default value:

{}

Type: attribute set of (attribute set)

Example value:

{
  add-inputs = {
    buildInputs = [pkgs.hello];
    TEST_ENV = "1";
    overrideAttrs = old: {
      nativeBuildInputs = old.nativeBuildInputs ++ [pkgs.pkg-config];
    };
  };
}

Declared in:

  • src/lib/namespacedConfigModule.nix

desktopFile

Desktop file

Type: null or string or (path) or (submodule)

Example value:

Set via a relative path to root:

{
  desktopFile = "./resources/app.desktop";
}

Set via a path:

{
  desktopFile = ./resources/app.desktop;
}

Set via an attrset:

{
  desktopFile = {
    name = "Test app";
    genericName = "App";
    comment = "An app that does stuff";
    icon = "./resources/icon.ico";
    categories = ["Category"];
  };
}

Declared in:
