NCI Documentation

These are options that can be set under package.metadata.nix. They are package specific. They can also be set under pkgConfig under nci.lib.makeOutputs:

{
  pkgConfig = common: {
    example-package = { /* options go here */ };
  };
}

app

Whether to enable app outputs.

Default value: false

Type: boolean

Example value: true

Declared in:

  • src/lib/namespacedConfigModule.nix

build

Whether to enable package outputs.

Default value: false

Type: boolean

Example value: true

Declared in:

  • src/lib/namespacedConfigModule.nix

depsOverrides

dream2nix overrides for the dependency derivation of this package.

Default value: { }

Type: attribute set of (attribute set)

Example value:

{
  add-inputs = {
    buildInputs = [pkgs.hello];
    TEST_ENV = "1";
    overrideAttrs = old: {
      nativeBuildInputs = old.nativeBuildInputs ++ [pkgs.pkg-config];
    };
  };
}

Declared in:

  • src/lib/namespacedConfigModule.nix

desktopFile

Desktop file

Default value: null

Type: null or string or path or (submodule)

Example value:

Set via a relative path to root:

{
  desktopFile = "./resources/app.desktop";
}

Set via a path:

{
  desktopFile = ./resources/app.desktop;
}

Set via an attrset:

{
  desktopFile = {
    name = "Test app";
    genericName = "App";
    comment = "An app that does stuff";
    icon = "./resources/icon.ico";
    categories = ["Category"];
  };
}

Declared in:

  • src/lib/namespacedConfigModule.nix

dream2nixSettings

Settings to pass to dream2nix.

Default value: [ ]

Type: list of (attribute set)

[{translator = "cargo-toml";}]

features

Set features to enable when building with a profile.

Default value: { }

Type: attribute set of list of string

Example value:

{
  release = ["default" "some-feature"];
  dev = ["default-dev"];
  bench = ["default" "bench"];
  test = ["default" "tests"];
}

Declared in:

  • src/lib/namespacedConfigModule.nix

profiles

Profiles to generate packages for. Set the profile to true to enable running tests, false to disable.

Default value: { dev = false; release = true; }

Type: attribute set of boolean

Example value:

{
  release = true;
  dev = false;
  custom-profile = false;
}

Declared in:

  • src/lib/namespacedConfigModule.nix
